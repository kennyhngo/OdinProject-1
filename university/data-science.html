Data Science Major at UofM

So, you're interested in the data science major, eh? I'll give you a brief run through of the clas requirements to get a data science major, and I will insight you on which classes I took and when I took them.

  1. Prerequisites
    • Calc 1 : MATH 115
    • Calc 2 : MATH 116
    • Calc 3 : MATH 215
    • Linear Algebra : MATH 214 or MATH 217
    • Elementary Programming Concepts : EECS 183
  2. Core Requirements
    • Computing and Discrete Mathematics
      • Discrete Mathematics : EECS 203
        or
        Intro to Combinatorics : MATH 465
      • Programming and Elementary Data Structures : EECS 280
    • Computing and Statistics
      • Data Structures and Algorithms : EECS 281
      • Introduction to Probability and Statistics : STATS 412
      • Applied Linear Regression : STATS 413
  3. Upper Level Courses
    • Machine Learning and Data Mining
      • Machine Learning : EECS 485
        • or
      • Data Mining : STATS 415
    • Database Management and Applications
      • Database Management Systems : EECS 484
        • or
      • Web Systems : EEECS 485
    • Applied Data Science Elective
  4. Technical Electives
    • Capstone Experience : STATS 485
    • 3 Elective Classes
